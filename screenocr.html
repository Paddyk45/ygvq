ScreenOCR

ScreenOCR is a lightweight Windows tool that lets you select part of your screen and instantly copy detected text to your clipboard.

No internet. No accounts. No background services. It runs, it copies, it leaves.
OVERVIEW
ScreenOCR automatically cleans up OCR output, but switches to raw text when it detects heavy symbol usage like math.

It is fully offline. Tesseract is bundled so nothing else is required.
DOWNLOAD
The executable is about 1.3 GB because Tesseract is fully bundled. Yes, it’s big. No, there is no smaller magic version right now.

KNOWN ISSUES
  • Startup may feel slow on first launch
  • OCR accuracy depends on font, resolution, and contrast
  • Very small selections may return no text
  • Complex layouts and images are not handled well
  • Math and symbol-heavy content may still be inaccurate
  • Windows only. Other platforms untested
  • Some antivirus software may flag it as a false positive
