features

overview of current tsugi modules
Combat +
  • Killaura — Automatically attacks nearby entities.
  • TargetStrafe — Circles enemies while attacking.
  • Criticals — Forces critical hits via packet manipulation.
  • Triggerbot — Attacks automatically when crosshair is on a target.
  • Velocity — Reduces or cancels knockback.
  • Reach — Extends attack distance.
  • AutoClicker — Clicks automatically at set CPS.
  • AutoTotem — Automatically equips Totem of Undying.
  • HitBoxes — Enlarges entity hitboxes.
  • BowAimbot — Automatically aims bows.
Movement +
  • Fly — Enables flight using multiple modes.
  • Speed — Boosts movement speed.
  • BHop — Bunny hopping for momentum.
  • LongJump — Extended jump distance.
  • Step — Walk up full blocks
  • NoSlow — Prevents slowdown while using items.
  • Jesus — Walk on water or liquids.
  • Spider — Climb walls.
  • FastFall — Fall faster than normal.
  • SafeWalk — Prevents walking off edges.
Render +
  • ESP — Shows entities through walls.
  • ChestESP — Highlights storage blocks.
  • XRay — Reveals valuable blocks.
  • Fullbright — Maximum brightness.
  • Nametags — Enhanced name tag rendering.
  • Tracers — Lines pointing to entities.
  • Chams — Solid-colored entity models.
  • NoRender — Disable selected visual effects.
Player +
  • AutoArmor — Automatically equips armor.
  • FastUse — Faster item usage.
  • NoFall — Prevents fall damage.
  • Regen — Faster health regeneration.
  • InventoryMove — Move while inventory is open.
  • AntiAFK — Prevents AFK kicks.
  • AutoEat — Eats when hungry.
World +
  • Scaffold — Places blocks automatically.
  • Nuker — Breaks blocks in an area.
  • ChestStealer — Quickly loots chests.
  • AutoMine — Mines selected blocks.
  • FastBreak — Break blocks faster.
  • Timer — Speeds up client tick rate.
Misc +
  • HUD — Custom on-screen display.
  • ClickGUI — Graphical module configuration.
  • Macros — Keybound actions.
  • AutoReconnect — Rejoin servers automatically.
  • ChatTweaks — Chat enhancements.
  • Notifications — On-screen alerts.
